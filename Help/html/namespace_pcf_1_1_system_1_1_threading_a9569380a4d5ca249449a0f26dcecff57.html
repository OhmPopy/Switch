enum Pcf::System::Threading::ThreadState
strong

Specifies the execution states of a System::Threading::Thread.

See Also
FileStream
Enumerator
Running 

The thread has been started, it is not blocked, and there is no pending System::Threading::ThreadAbortException.

StopRequested 

The thread is being requested to stop. This is for internal use only.

SuspendRequested 

The thread is being requested to suspend.

Background 

The thread is being executed as a background thread, as opposed to a foreground thread. This state is controlled by setting the System::Threading::Thread.IsBackground property.

Unstarted 

The System::Threading::Thread.Start() method has not been invoked on the thread.

Stopped 

The thread has stopped.

WaitSleepJoin 

The thread is blocked. This could be the result of calling System::Threading::Thread.Sleep(System::Int32) or System::Threading::Thread.Join(), of requesting a lock ï¿½ for example, by calling System::Threading::Monitor.Enter(System::object) or System::Threading::Monitor.Wait(System::object,System::Int32,System::Boolean) or of waiting on a thread synchronization object such as System::Threading::ManualResetEvent.

Suspended 

The thread has been suspended.

AbortRequested 

The System::Threading::Thread.Abort(System::object) method has been invoked on the thread, but the thread has not yet received the pending System::Threading::ThreadAbortException that will attempt to terminate it.

Aborted 

The thread state includes System::Threading::ThreadState.AbortRequested and the thread is now dead, but its state has not yet changed to System::Threading::ThreadState.Stopped.
