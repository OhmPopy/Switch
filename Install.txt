Mac Os X
--------

  1. Install CMake

     in order to build Pcf in your favvorite IDE, you first need download and install CMake from https://cmake.org/
     Remarks: The minimum version of used is 3.5.

  2. Install Homebrew

     After cmake, to build Pcf, you need to install the latest version curl, jpeg, pnglib and zlib. 
     The easiest way to get this is from Homebrew (http://brew.sh/).

  3. After installing brew, do the following:

     ~$ brew update
     ~$ brew install curl
     ~$ brew install jpeg
     ~$ brew install libpng
     ~$ brew install zlib

  4. Create cmake folder in usr/local with administrator privilege:

     ~$ sudo mkdir /usr/local/cmake

  5. Create ThirdParties project for your prefered ide or unix makefile (in this case Xcode) with CMake gui or with
     the next command lines.
     Remarks: type "cmake --help" for more information.
    
     ~$ mkdir Build
     ~$ cd Build
     ~$ mkdir ThirdParties
     ~$ cd ThirdParties
     ~$ cmake -G "Xcode" ../../ThirdParties
     
  6. Bu!ld and install in Xcode or like this:
  
     ~$ xcodebuild -target install -configuration Release
  
  7. Create Pcf project for your prefered ide or unix makefile (in this case Xcode) with CMake gui or with
     the next command lines.
     Remarks: type "cmake --help" for more information.
    
     ~$ cd ..
     ~$ cmake -G "Xcode" ..
     
  8. Bu!ld and install in Xcode or like this:
  
     ~$ xcodebuild -target install -configuration Release
  
  9. Enjoy and be creative with Pcf ;-)
