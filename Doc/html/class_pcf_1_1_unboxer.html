Detailed Description

Support unboxing of type You can use this when you want convert a boxed type to an unboxed type (e.g. Int32 to int32)

Boolean ready = true;
Console::WriteLine("ready = {0}", ready); // console output : ready = true
Console::WriteLine("ready = {0}", Unboxer::Unboxing(ready)); // console output : ready = 1
