class  Pcf::Property< T, Attribute >
 A property is a member that provides a flexible mechanism to read, write, or compute the value of a private field. Properties can be used as if they are public data members, but they are actually special methods called accessors. This enables data to be accessed easily and still helps promote the safety and flexibility of methods. This sample shows a Person class that has two properties: Name (string) and Age (int).
