The Pcf.Core Examples

The Pcf.Core examples provide more informations for manipuling Pcf.Core features.

Hello World

  • HelloWorld The classic first application.

Main

  • Main shows how to use Pcf::System::Environment::SetCommandLineArgs, Main and params.

Object

  • Object usage deemonstrates how to inherit from Pcf::System::Object.
  • Object Equals show how to use Pcf::System::Object Equals method.
  • Object GetType show how to use Pcf::System::Object GetType method.
  • Object MemberwiseClone show how to use Pcf::System::Object MemberwiseClone method.
  • Object ReferenceEquals show how to use Pcf::System::Object ReferenceEquals method.
  • Object ToString show how to use Pcf::System::Object ToString method.

String

  • Create String demonstrates different methods to create a Pcf::Pcf::System::String.
  • String Enumerator show how to enumerate a Pcf::System::String.

Array

  • Create Array demonstrates different methods to create an Pcf::System::Array.
  • Using Array creates and initializes an Pcf::System::Array and displays its properties and its elements.
  • Array Length show how to get the Pcf::System::Array Length property.
  • Array Operator shows how to use operator [] to list the elements of an Pcf::System::Array.
  • Array Functor shows how to use operator () to list the elements of an Pcf::System::Array.
  • Array IndexOf show how to used Pcf::System::Array IndexOf method.
  • Array LowerBound show how to get the Pcf::System::Array GetLowerBound method.
  • Array Enumerator show how to enumerate an Pcf::System::Array.
  • Array SyncRoot show how to lock an Pcf::System::Array with SyncRoot property.

Enum

  • Enum usage show how to inherit from Pcf::System::Enum.
  • FlagsEnum usage show how to inherit from Pcf::System::FlagsEnum.

foreach

  • Foreach C Array show how to use #pcf_foreach on C array.
  • Foreach List show how to use #pcf_foreach on List< T >.
  • Foreach String show how to use #pcf_foreach on Pcf::System::String.
  • Foreach on Tuples show how to use #pcf_foreach on Pcf::System::Tuple <T1, T2, T3> collection.

System

  • Environment show how to use Pcf::System::Environment class.
