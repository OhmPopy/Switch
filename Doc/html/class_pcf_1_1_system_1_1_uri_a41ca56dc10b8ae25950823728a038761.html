String Pcf::System::Uri::GetLeftPart ( UriPartial  part) const
inline

Gets the specified portion of a Uri instance.

Parameters
part= One of the UriPartial values that specifies the end of the URI portion to return.
Returns
string A string that contains the specified portion of the Uri instance.
Exceptions
InvalidOperationExceptionThis instance represents a relative URI, && this property is valid only for absolute URIs.
ArgumentExceptionThe specified part is ! valid.
Remarks
The GetLeftPart method returns a string containing the leftmost portion of the URI string, ending with the portion specified by part.
GetLeftPart includes delimiters in the following cases:
* Scheme includes the scheme delimiter.
* Authority does ! include the path delimiter.
* Path includes any delimiters in the original URI up to the query || fragment delimiter.
* Query includes the Path, plus the query && its delimiter.
