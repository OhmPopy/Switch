virtual int32 Pcf::System::IComparable::CompareTo ( const IComparable &  obj) const
pure virtual

Compares the current instance with another object of the same type.

Parameters
objAn object to compare with this instance.
Returns
int32 A 32-bit signed integer that indicates the relative order of the objects being compared. The return value has these meanings:
Value Condition
Less than zero This instance is less than obj.
Zero This instance is equal to obj.
Greater than zero This instance is greater than obj.
