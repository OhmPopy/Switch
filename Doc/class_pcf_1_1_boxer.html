Detailed Description

Support boxing of type You can use this when you want convert an unboxed type to a boxed type (e.g. int32 to Int32)

bool ready = true;
Console::WriteLine("ready = {0}", ready); // console output : ready = 1
Console::WriteLine("ready = {0}", Boxer::Boxing(ready)); // console output : ready = true
